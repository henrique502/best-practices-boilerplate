# Best Practices Boilerplate

Este Power fornece steering files com convenções e boas práticas para projetos TypeScript/React.

## O que está incluído

- **TypeScript Best Practices** — tipagem estrita, padrões de código, tratamento de erros
- **React Best Practices** — componentes funcionais, hooks, performance
- **Security Best Practices** — OWASP, gestão de segredos, proteção de dados
- **Git Best Practices** — conventional commits, branching, workflow
- **Shell Environment** — uso de Git Bash com sintaxe Unix

## Como usar

Instale este Power no seu workspace e os steering files serão automaticamente aplicados nas interações com o Kiro.
